Projetos em Destaque

Soluções reais desenvolvidas para otimizar processos e resolver problemas.

Chamaí

Sistema completo de gestão de chamados (Ticketing). Centraliza solicitações de suporte de TI, permitindo acompanhamento de status, lançamento de histórico para documentação de resolução de problemas, priorização e relatórios de atendimento, atendimento por máquinas, SLA e performance dos colaboradores. Além disso, o sistema dispara e-mails transacionais para notificar os solicitantes das atualizações nos chamados.

React Node.js PostgreSQL

Autbuy

Robô de automação para o setor de compras. O sistema recebe a listagem de estoque e vendas e com base nisso e nos critérios definidos pelo usuário, sugere a quantidade a ser comprada, aguardando aprovação do analista de compras antes de fazer o envio da cotação. Após receber os dados de estoque e vendas, salva em um banco para aplicar machine learning e melhorar a precisão das compras.

N8N Automação Banco de Dados

Checklist Inteligente

Sistema de validação diária de balanças com tolerância de peso automática. O diferencial é o fluxo no N8N que monitora o preenchimento: dispara alertas (Email) às 07:30 se pendente, e escala para a gerência às 08:00. Caso o ususário não faça o checklist durante o dia de trabalho, o sistema preenche como 'não realizado' para auditoria posteriormente.
